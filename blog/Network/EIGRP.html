EIGRP (Enhanced Interior Gateway Routing Protocol)

Giriş

EIGRP — Cisco tərəfindən hazırlanmış, IGP (Interior Gateway Protocol) kateqoriyasına daxil olan dinamik marşrutlaşdırma protokoludur. O, həm distance vector, həm də link-state metodlarının üstünlüklərini birləşdirərək "advanced distance-vector" protokolu kimi tanınır.

Əsas Xüsusiyyətlər

  • DUAL (Diffusing Update Algorithm): Yönləndirmə döngələrindən qorunmaq və tez konvergensiya üçün istifadə olunur.
  • Classless Routing: Subnet mask dəstəyi var.
  • Incremental Updates: Yalnız dəyişən marşrut məlumatları ötürülür.
  • Load Balancing: Eyni anda bir neçə bərabər və ya fərqli metric-lərlə marşrut seçimi mümkündür.

Metric Hesablanması

EIGRP metric hesablamasında aşağıdakı parametrlərdən istifadə edir:

  • Bandwidth
  • Delay
  • Reliability
  • Load
  • MTU (amma hesablamada istifadə olunmur)

EIGRP Konfiqurasiyası (Cisco CLI)


Router(config)# router eigrp 100
Router(config-router)# network 192.168.1.0 0.0.0.255
Router(config-router)# network 192.168.2.0 0.0.0.255
Router(config-router)# no auto-summary

Terminlər

  • AS (Autonomous System): Eyni EIGRP domenində olan router-ları qruplaşdırmaq üçün istifadə olunur.
  • Successor: Ən optimal marşrut.
  • Feasible Successor: Backup marşrut.
  • Neighbor: Hello mesajları ilə əlaqə saxlayan qonşu router-lar.

Nəticə

EIGRP, Cisco əsaslı şəbəkələrdə sürətli və effektiv dinamik marşrutlaşdırma təklif edir. Təhlükəsizlik, performans və idarəetmə baxımından stabil və güvənilirdir.
